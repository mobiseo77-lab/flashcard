# 📚 암기카드 PWA - 설치 가이드

## 🚀 무료 배포 방법 (GitHub Pages 추천)

### 방법 1: GitHub Pages (무료, 가장 쉬움)

1. **GitHub 계정 만들기** → https://github.com/signup

2. **새 저장소(Repository) 만들기**
   - 이름: `flashcard` (아무 이름 가능)
   - Public 선택
   - Create repository 클릭

3. **파일 업로드**
   - 이 ZIP 안의 파일들을 모두 업로드
   - `index.html`, `manifest.json`, `sw.js`, `icons/` 폴더 전부

4. **GitHub Pages 활성화**
   - Settings → Pages → Source: `main` 브랜치 선택 → Save

5. **주소 확인**
   - 약 1~2분 후 `https://사용자명.github.io/flashcard` 주소로 접속 가능

---

### 방법 2: Netlify Drop (드래그앤드롭, 초간단)

1. https://app.netlify.com/drop 접속
2. 이 ZIP 파일을 그대로 드래그앤드롭
3. 자동으로 URL 생성됨 (예: `https://wonderful-abc123.netlify.app`)

---

## 📱 스마트폰 홈화면에 설치하기

### 안드로이드 (Chrome)
1. 위 URL을 Chrome으로 접속
2. 주소창 오른쪽 메뉴(⋮) → **"홈 화면에 추가"** 또는 **"앱 설치"**
3. 홈화면에 아이콘 생성 완료!

### 아이폰 (Safari)
1. Safari로 접속 (Chrome 아님!)
2. 하단 공유 버튼(□↑) 탭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"** 탭

---

## ✨ 특징
- 📴 **오프라인 지원**: 한번 접속 후 인터넷 없어도 사용 가능
- 💾 **자동 저장**: 종료 시 현재 상태 저장, 재시작 시 이어서 학습
- 🔍 **문자 인식(OCR)**: 이미지에서 텍스트 자동 추출
- 📸 **이미지 카드**: 교재 사진을 그대로 카드로 등록
- 🎓 **라이트너 박스**: 3단→2단→1단 체계적 암기 시스템
